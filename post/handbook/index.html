有用的命令

2010-07-17 1 min read # 命令 # shell

查看端口和连接信息

netstat -ntlp

列出占用端口进程

lsof -i :8080

查看iptables状态

/etc/init.d/iptables status

添加8080端口

/sbin/iptables -I INPUT -p tcp --dport 8080 -j ACCEPT
/etc/rc.d/init.d/iptables save
/etc/init.d/iptables restart

按日期动态命名日志文件

/`date "+%F"`.`date "+%H-%M-%S"`.`date "+%N"` 2>&1
