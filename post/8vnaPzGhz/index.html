java将html转图片，使用html2image

2020-12-06 1 min read # html2image # html转图片 # java 
<dependency>
    <groupId>gui.ava</groupId>
    <artifactId>html2image</artifactId>
    <version>2.0.1</version>
</dependency>

注意版本使用最新版 https://mvnrepository.com/artifact/gui.ava/html2image

String s = "your html";
Html2Image html2Image = Html2Image.fromHtml(s);
ImageRenderer imageRenderer = html2Image.getImageRenderer();
imageRenderer.saveImage("1.png");

在存储为png时候，没有背景色，并且表格有实线
