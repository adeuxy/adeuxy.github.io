PowerDesigner中comment字段和name字段相互转换#数据库

PowerDesigner导入sql

版本:1.0    发布时间:2014-10-23 19:29:12    分类:数据库

    

    1.PowerDesigner支持导入sql来建表，逆向sql生成表

    打开File-->Reverse Engineer-->Database

    选择对应的DBMS

    选择Using script files-->OK

    生成的工程中表没有name信息；


    sql语句中只有column字段，而PowerDesigner用name字段显示字段名称

   2.将sql中的comment复制到name

        ctrl + shift + x 执行脚本 comment2name.vbs(点击下载)

        


  3.将sql中的name复制到comment

        name2comment.vbs(点击下载)
