centos安装jdk,tomcat,mysql构建java开发环境

1.下载jdk1.8，推荐使用，1.8解决了resize图片时格式不支持的问题

下载地址：http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

jdk-8u25-linux-x64.rpm，对应系统平台

查看当前系统平台：

    uname -a

查看当前系统版本：

    cat /etc/redhat-release

这里是：CentOS release 6.3 (Final)

安装：

    rpm -ivh jdk-8u25-linux-x64.rpm

安装后查看java：

    java -version

2.下载tomcat，这里用tomcat8

在服务器直接使用wget下载：

wgt http://mirrors.hust.edu.cn/apache/tomcat/tomcat-8/v8.0.14/bin/apache-tomcat-8.0.14.tar.gz

解压

    tar -zxvf apache-tomcat-8.0.14.tar.gz .

后进入bin目录执行

    ./start.sh

使用 tail -f logs/catalina.out 看日志    

之后通过浏览器访问http://服务器ip地址:8080/看到tomcat首页

3.下载mysql，虽然yum有，但是版本较低，这里安装社区版5.6，Linux - Generic (glibc 2.5) (x86, 64-bit), RPM Bundle

在服务器直接使用wget下载：

wgt http://cdn.mysql.com/Downloads/MySQL-5.6/MySQL-5.6.21-1.linux_glibc2.5.x86_64.rpm-bundle.tar

Bundle是合集，解压后进入目录 

rpm -ivh * 

安装后mysql初始账号是root，密码在/root/.mysql_secret

修改密码：mysqladmin -uroot -p初始密码 password新密码

创建用户：CREATE USER 用户名 IDENTIFIED BY '密码'

赋予用户远程连接权限：GRANT ALL PRIVILEGES ON *.* TO '用户名'@'%' IDENTIFIED BY '密码' WITH GRANT OPTION

最后：flush privileges
