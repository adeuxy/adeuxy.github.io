centos安装jdk,tomcat,mysql构建java开发环境

1.下载jdk1.8，推荐使用，1.8解决了resize图片时图片格式不支持的问题

下载地址：http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

jdk-8u25-linux-x64.rpm，对应系统平台

查看当前系统平台：

    uname -a

查看当前系统版本：

    cat /etc/redhat-release

安装：

    rpm -ivh jdk-8u25-linux-x64.rpm

安装后查看java：

    java -version

2.下载tomcat，这里用tomcat8

在服务器直接使用wget下载：

wgt http://mirrors.hust.edu.cn/apache/tomcat/tomcat-8/v8.0.14/bin/apache-tomcat-8.0.14.tar.gz
